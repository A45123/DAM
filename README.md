# 🎮 Desenvolvimento de Aplicações Móveis

## 🎯 Objetivos de Aprendizagem

Ao concluir esta Unidade Curricular, os estudantes serão capazes de:

1. Conhecer e compreender os mecanismos para desenvolvimento de apps móveis.  
2. Utilizar tecnologias-chave para desenvolvimento de aplicações móveis.  
3. Aplicar metodologias centradas na experiência do utilizador em jogos digitais.  
4. Desenvolver pensamento crítico sobre tecnologias e metodologias aplicadas.  
5. Criar um jogo funcional para dispositivos móveis.
